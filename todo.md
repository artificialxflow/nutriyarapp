# 📋 لیست وظایف پروژه رژیمیار (NutriYar)

این یک چکلیست عملیاتی برای طراحی و پیادهسازی فرانتاند اپلیکیشن رژیمیار است. تمرکز بر طراحی مدرن، ریسپانسیو و راستچین (RTL) با استفاده از Tailwind CSS است.

---

## 🏗️ فاز ۱: زیرساخت و آمادهسازی (Foundation)
- [x] ایجاد ساختار پوشهبندی پروژه (Assets, Components, Pages)
- [x] کانفیگ Tailwind CSS برای پشتیبانی از جهت **RTL**
- [x] انتخاب و نصب فونت فارسی مدرن (مانند `Vazirmatn`)
- [x] تعریف پالت رنگی اختصاصی در فایل کانفیگ Tailwind:
    - [x] رنگ سبز سالم: `#22c55e`
    - [x] رنگ سبز تیره: `#16a34a`
    - [x] رنگ نارنجی انرژی: `#f97316`
- [x] راهاندازی کتابخانه آیکنها (Lucide)
- [x] ایجاد استایلهای پایه برای کارتها (Border مشخص، سایه نرم، گوشه گرد `rounded-2xl`)

---

## 🌐 فاز ۲: طراحی لندینگ پیج (Landing Page)
- [x] **هدر (Header):** طراحی لوگو، منوی ناوبری و دکمههای ورود/ثبتنام (چسبان - Sticky)
- [x] **بخش هیرو (Hero Section):**
    - [x] قرار دادن تیتر جذاب و دکمه فراخوان (CTA)
    - [x] بهینهسازی تصویر غذای سالم برای نمایش ریسپانسیو
- [x] **بخش قابلیتها (Features):** پیادهسازی ۶ کارت معرفی خدمات با آیکنهای رنگی
- [x] **بخش رسپیهای محبوب:** طراحی اسلایدر یا گرید کارتهای غذا (کالری، زمان، تصویر)
- [x] **بخش نحوه کار (How it works):** طراحی گامهای ۱ تا ۳ به صورت بصری
- [x] **نظرات کاربران (Testimonials):** طراحی کارتهای نقلقول همراه با آواتار
- [x] **فوتر (Footer):** بخش لینکهای سریع، شبکههای اجتماعی و کپیرایت

## 📱 فاز ۳: پنل کاربری موبایلمحور (User Dashboard)
*هدف: طراحی با تجربه کاربری شبیه به اپلیکیشن Native اندروید*

- [x] **ناوبری پایین (Bottom Navigation):** طراحی منوی ثابت با آیکنهای خانه، غذاها، برنامه، گزارش و پروفایل
- [x] **داشبورد اصلی (Home):**
    - [x] طراحی ویجت دایرهای پیشرفت کالری مصرفی
    - [x] طراحی بخش ثبت آب مصرفی (Interactive-like UI)
    - [x] کارت نمایش وزن فعلی و هدف
- [x] **صفحه ثبت غذا (Log Food):**
    - [x] طراحی فیلد جستجوی مدرن
    - [x] لیست کارتهای نتایج جستجو با جزئیات ماکرومغذیها (پروتئین، کربوهیدرات، چربی)
- [ ] **صفحه رسپیها (Recipes):** گرید دو ستونه از غذاها با فیلتر دستهبندی
- [ ] **صفحه برنامه غذایی (Meal Plan):** طراحی تایملاین روزانه (صبحانه تا شام)
- [ ] **صفحه پروفایل:** فرمهای نمایش قد، وزن و هدف با استایل Clean UI
